Close Button

¿Qué es?

Es un botón de cierre estandarizado para cerrar componentes como modales, alertas y offcanvas.

Características

  • Ícono visual sin texto
  • Variantes para fondos oscuros
  • Accesible con aria-label
  • Personalizable con CSS
{{ alert }}
Clases principales
Clase Función
.btn-close Botón de cierre básico de Bootstrap
.btn-close-white Variante para fondos oscuros (obsoleta desde v5.3)
Atributos importantes
Atributo Función
data-bs-dismiss="modal" Cierra un modal
data-bs-dismiss="offcanvas" Cierra un offcanvas
data-bs-dismiss="alert" Cierra una alerta
aria-label="Close" Mejora la accesibilidad
Dato
No necesita texto porque Bootstrap ya incluye el ícono SVG internamente.
