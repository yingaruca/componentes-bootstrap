Navbar

El Navbar es un componente de navegación de Bootstrap que permite organizar enlaces, logos y menús dentro de una página web. Es uno de los componentes más usados porque permite navegar entre secciones de la página o entre diferentes páginas del sitio. Su principal objetivo es ofrecer una navegación clara y accesible en cualquier dispositivo.

Características principales
  • Responsivo: se adapta automáticamente a pantallas pequeñas.
  • Menú hamburguesa: en móviles, el contenido se colapsa y aparece un botón para desplegarlo.
  • Soporte para marca: permite incluir un logo o nombre del sitio.
  • Dropdowns: se pueden añadir menús desplegables dentro de la navegación.
  • Formularios integrados: como buscadores o botones.
  • Personalización de colores: mediante clases como .navbar-light, .navbar-dark, .bg-dark, .bg-primary.
  • Compatibilidad con contenedores: se suele usar junto a .container o .container-fluid.

Clases principales
Clase Función
.navbar Contenedor principal del componente
.navbar-brand Logo o nombre de la marca
.navbar-toggler Botón hamburguesa para pantallas pequeñas
.navbar-collapse Contenido que se colapsa en móviles
.navbar-nav Lista de elementos de navegación
.nav-item Elemento individual dentro de la lista
.nav-link Enlace dentro del Navbar
.dropdown Contenedor para menús desplegables
.dropdown-menu Lista del menú desplegable
.dropdown-item Elemento dentro del dropdown
